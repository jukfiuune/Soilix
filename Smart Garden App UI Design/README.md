
  # Smart Garden Native

  This project has been rewritten from a React web/Vite prototype into a React Native app powered by Expo.

  ## Running the app

  Run `npm install` to install dependencies.

  Run `npm run start` to launch the Expo development server.

  Optional platform commands:

  - `npm run android`
  - `npm run ios`
  - `npm run web`
